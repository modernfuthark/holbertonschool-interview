<!-- Task -->
> Given an array of useable coins, find the minimum amount of coins to reach a total.

<!-- Prototype -->
`makeChange(coins, total)`

<!-- Unique return values -->
If `total` cannot be reached, return `-1`.\
If `total` is 0 or less, return `0`.
